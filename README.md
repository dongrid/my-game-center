# 🕹️ My Game Center

Streamlitで作られた、6つのゲームが楽しめるゲームセンターです！

## 🎮 収録ゲーム

### 1. 🎮 数当てゲーム
1から100（または1000）の間でランダムに選ばれた数字を当てるゲームです。
- **Level 1**: 1〜100の範囲
- **Level 2**: 1〜1000の範囲
- 相棒キャラがヒントをくれる！

### 2. ✊ じゃんけんバトル
CPUとじゃんけんで対戦！3回勝つとバルーンが飛びます。
- 勝敗記録が残る
- 連勝するとお祝いが！

### 3. 👾 インベーダー・クエスト PRO
迫りくる敵を撃ち落とすシューティングゲーム。
- **高度なスコアシステム**: 命中率、タイム、ショット数を総合評価
- 正確に、かつ速くクリアするほど高得点！

### 4. 👥 クローン・シュート
ゲートを通って軍勢を増やし、敵を倒すアクションゲーム。
- ゲートで兵士を増やす（+、×、-）
- 兵士の数だけ弾が発射される！
- 敵の残りHP分だけ味方が減る緊張感

### 5. 🏃 ブロック・ランナー
マリオ風の横スクロールアクションゲーム。
- 左右キーで移動、スペースキーでジャンプ
- 敵を踏んで倒せる
- ゴールを目指せ！

### 6. 🏎️ マリオカート風レーシング
レーンを移動してライバル車を避けるレーシングゲーム。
- 4レーン制の道路
- コインを集めてスコアアップ
- パワーアップアイテムでスピードアップ！

## 🚀 セットアップ方法

### 1. リポジトリのクローン
```bash
git clone https://github.com/あなたのユーザー名/my-game-center.git
cd my-game-center
```

### 2. 依存関係のインストール
```bash
pip install -r requirements.txt
```

### 3. アプリの起動
```bash
streamlit run app.py
```

ブラウザが自動的に開き、ゲームセンターが表示されます！

## 🌐 公開方法（Streamlit Cloud）

1. このリポジトリをGitHubにプッシュ
2. [Streamlit Community Cloud](https://share.streamlit.io/) にアクセス
3. GitHubアカウントでログイン
4. 「New app」をクリック
5. リポジトリと `app.py` を選択
6. 「Deploy!」をクリック

数分で世界中に公開されます！
